componentes electrónicos básicos

Un componente electrónico es una pieza fundamental usada en la construcción de circuitos electrónicos. Cada componente tiene una función específica, como controlar la corriente, almacenar energía, emitir señales o procesarlas.

Resistencia

Función: Limita o regula el flujo de corriente eléctrica en un circuito.!

Unidad: Ohmios (Ω)

Condensador

Función: Almacena energía en forma de campo eléctrico; se usa para filtrar, estabilizar o temporizar señales.

Unidad: Faradios (F)

Inductor

Función: Almacena energía en forma de campo magnético; común en filtros, transformadores y fuentes de poder.

Unidad: Henrios (H)

Diodo

Función: Permite el paso de corriente en una sola dirección. Se usa para rectificación de corriente alterna a continua.

Variante común: Diodo LED (emite luz)

5. LED

Función: Emite luz cuando pasa corriente por él. Se usa en iluminación e indicadores.

Símbolo: |>| con flechas indicando luz
